# 프로젝트 개요
Python 기본문법을 사용하여 간단한 퀴즈 프로그램을 만들어봅니다.  
JSON파일을 이용하여 데이터 영속성을 구현해보고, Git으로 변경 이력을 관리해봅니다.
# 퀴즈 주제 선정 이유
제가 응원하는 LCK팀인 T1으로 애정을 담아 문제를 구성하였습니다.
# 실행 방법
Python Version : 3.10 이상
```bash
python main.py
```
# 기능 목록
|기능|내용|
|------|------------------|
|퀴즈 풀기|저장된 문제를 풀고 점수 확인|
|퀴즈 추가|새로운 문제를 입력 받아 저장|
|퀴즈 목록|저장된 모든 문제를 확인|
|점수 확인|최고 점수 확인|
|예외처리|빈 입력, 문자 입력, 범위 초과, Ctrl+C, EOFError 모두 처리|
|데이터처리|state.json이 없거나 손상된 경우 기본 퀴즈 데이터로 자동 초기화|

# 파일 구조
```
project/
├── main.py             # 프로그램 진입점
├── quiz.py             # Quiz 클래스 정의
├── game.py             # Game 클래스 정의
├── state.json          # 퀴즈 데이터 및 점수 저장 파일
├── screenshots/        # 실행 화면 스크린샷
├── .gitignore 
└── README.md           #프로젝트 설명서
```
# 데이터 파일 설명
경로 : 루트 폴더의 ```state.json```  
역할 : 퀴즈 목록과 최고 점수를 저장  
스키마 :  
```json
{
    /* 등록된 퀴즈 목록 */
    "quizzes": [ 
        {   /* 문제, 선택지, 정답 */
            "question": "LCK 팀 T1의 이전 팀명은?",
            "choices": [...],
            "answer": 1
        },
        ...
    ],
    /* 최고 점수 */
    "best_score": 3,
}
```
